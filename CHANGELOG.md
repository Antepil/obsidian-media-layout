# Changelog

## 0.1.2

- Keep the Live Preview media widget visible when the cursor is on a media line.
- Enlarge the resize hit area so dragging the right edge or lower-right corner works reliably.
- Prevent image clicks inside plugin layouts from opening Obsidian's native image preview.
- Fix Reading View media extraction so embed containers and nested images are not handled twice.

## 0.1.1

- Rename the plugin display name to Media Intelligent layout.
- Keep the plugin id as `media-layout` for BRAT and existing Obsidian installs.

## 0.1.0

- Initial beta release.
- Add automatic image/video grouping for native Obsidian embeds.
- Add Live Preview resize handles for single media and media groups.
- Add Reading View layout rendering.
- Add settings for layout gap, max columns, minimum media width, fill mode, and persistence.
